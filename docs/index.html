About the n8n Workflow Catalog

This catalog contains ready-to-import n8n workflow automations that leverage the People.ai MCP (Model Context Protocol) integration alongside AI models, CRM data, and communication tools to deliver actionable sales intelligence.

Each workflow is a complete n8n automation that can be imported directly into your n8n instance. Workflows combine People.ai's engagement data with Claude AI to generate insights like daily sales digests, meeting briefs, pipeline coaching reports, and account risk alerts.

How it works:
1. Browse the catalog and pick a workflow
2. Download the workflow JSON (Quick Start or Full version)
3. Import into your n8n instance
4. Connect your credentials (People.ai MCP, Anthropic, Slack, etc.)
5. Configure schedule and channel settings
6. Activate the workflow

Prerequisites:
• An n8n instance (self-hosted or n8n Cloud)
• People.ai MCP credentials
• Anthropic API key for Claude AI
• Slack Bot token (for most workflows)
• Additional credentials as noted per workflow
